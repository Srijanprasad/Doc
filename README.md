# Srijan Prasad - Portfolio

A personal portfolio website showcasing my projects, skills, certifications, and work in web development.

## 🚀 Live Demo
https://github.com/Srijanprasad

## 🛠 Tech Stack

- **Frontend:** React, Vite  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Routing:** React Router  
- **API / Integration:** EmailJS  
- **Build Tools:** Vite  
- **Version Control:** Git, GitHub  
- **Deployment:** Vercel  

## ✨ Features

- Responsive and modern UI  
- Smooth animations and transitions  
- Projects, certifications, and experience sections  
- Tech stack showcase  
- Contact form integration  

## 📂 Setup

```bash
git clone https://github.com/Srijanprasad/Portfolio.git
cd Portfolio
